# 문제

문제 설명
정수를 담고 있는 배열 arr의 평균값을 return하는 함수, solution을 완성해보세요.

제한사항
- arr은 길이1이상,100이하인 배열입니다.
- arr의 원소는 -10000 이상 10000이하인 정수입니다.
입출력 예
arr | return 
---|---
[1,2,3,4] | 2.5 
[5,5] | 5
